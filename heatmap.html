  • Легче, проще, быстрее!

    Весь JS и CSS код новой версии API в два раза меньше, чем в старой. Это кардинально увеличило скорость загрузки страниц с картой.

    Теперь разработчики могут самостоятельно выбирать модули для загрузки. Используйте именно то, что нужно вашему проекту!

  • Модульность

    Новый API совместим с большинством модулей библиотеки Leaflet. Кластеризация, тепловые карты, анимированные маркеры и многие другие. Не нашли нужного? Создайте свой — теперь это просто, как никогда ранее!

  • Новый дизайн и выбор темы

    Используйте тему, которая будет лучше смотреться на вашем сайте — тёмную или светлую.
