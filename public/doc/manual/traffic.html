DG.Traffic

  • Конструктор
  • Опции

Создает слой с информацией о пробках. При наведении на участок дороги отображает дополнительную метаинформацию.

Конструктор

Конструктор Использование Описание
DG.Traffic( <DG.Traffic options> options? ) DG.Traffic(…) Создает слой пробок.

Опции

Опция Тип По умолчанию Описание
disableLabel Boolean false Если передать значение true, при наведении на участок дороги с пробками не будет отображаться хинт с метаинформацией.
