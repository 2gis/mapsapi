DG.template

Используется для шаблонизации в Javascript с возможностью задавать условные операторы и циклы. Основан на функции Джона Ресига

Функция-конструктор

Использование Возвращает Описание
DG.template(String template, Object data) String html Считывает шаблон и данные для подстановки и возвращает строку с итоговым html кодом

Опции

Опция Тип Описание
template String Шаблон
data Object Данные для подстановки в шаблон
