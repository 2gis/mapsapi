  • Легковесность

    JS и CSS код новой версии API занимает всего 105 Кб. в базовой поставке, вместо прежних 255 Кб., что позволило увеличить скорость открытия страниц с картой.

  • Модульность

    Новый API совместим с большинством модулей библиотеки Leaflet. Помимо готовых модулей, таких как кластеризация, тепловая карта или анимированные маркеры, вы можете добавить свои.

  • Новый дизайн и выбор темы

    Используйте тему, которая будет лучше смотреться на вашем сайте — тёмную или светлую.
